Anchor Elements and HyperLinks

-HTML Atrributes
<tag attribute=value>Content</tag>

-To add more attributes, add a space in between
<tag attribute=value anotherAttribute=Value>Content</tag>

-Example
<a>This is a link</a>
-code wont work as a link

-Example
<a href="https://www.google.com/">This is a link</a>
-now the link will work in code
